# 🧳 出國旅遊小工具 Travel Kit

一個檔案搞定出國：**行前規劃 → 旅途隨身 → 回顧結算**。
單檔 HTML、免安裝、離線可用（即時匯率等連網功能為加分項）。

👉 **[直接開啟使用](index.html)**｜**[操作說明](guide.html)**

## 功能
- 🗓 行程總表（時段制、待規劃標記、支援把零散資料交給 Claude 排好後 JSON 匯入）
- ✈️ 訂票＆住宿總覽（已訂／待訂／比較中、確認編號、當地語言地址）
- 🧳 三段式打包（待買 → 分人打包模板 → 出門前檢查＋出發交通提醒）
- 💴 記帳分帳（當地幣自動換台幣、分類 chips、誰付誰參與、自動算誰給誰多少）
- 🎁 伴手禮託買、📶 網卡漫遊、🛂 入境文件 QR、🆘 緊急資訊卡（日本官方電話預填）
- 📊 回顧統計（每日花費、分類佔比、每人結算）＋唯讀分享頁匯出

## 隱私
所有資料只存在**你自己瀏覽器**（localStorage），不會上傳到任何伺服器。
換裝置請用「設定 → 匯出／匯入備份 JSON」。

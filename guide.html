🧳 出國旅遊小工具・操作說明

一個檔案搞定：行前規劃 → 旅途隨身 → 回顧結算。給自己與同行家人朋友用。

🚀 快速上手（3 步驟）

  1. 打開 index.html → 在「總覽」建立旅程：名稱、日期、成員（例：我, 媽, 妹）。
  2. 行前：排「行程」、記「訂票住宿」、勾「打包」。
  3. 旅途中：總覽看「今天」、隨手「快速記帳」；回來看「統計」直接算好誰給誰多少。

📱 手機怎麼用（重要）

把 index.html 存到手機的「檔案」App，用 Safari／Chrome 直接開啟再使用。

⚠️ 資料存在「開啟它的那個瀏覽器」裡。從 LINE 的內建視窗打開，資料可能不會保留——請先存到檔案 App 再開。換裝置或怕遺失，就用「設定 → 匯出備份 JSON」。

🗓 行程

  • 每天分「上午／下午／晚上」，有預約的再填精確時間；沒排的天會標「待規劃」。
  • 請 Claude 幫你排：按「複製指令」→ 貼到 Claude 對話，附上你的零散資料（訊息、備忘錄）→ 把 Claude 回的 JSON 貼進「匯入」。格式範例：
[{"date":"2026-08-01","slot":"morning","time":"",
  "name":"淺草寺","place":"東京都台東區","map":"https://…",
  "trans":"銀座線 淺草站","cost":0,"book":"","note":""}]

✈️ 訂票住宿＋附件約定

  • 每筆有狀態：已訂／待訂／比較中——還沒決定的也先記一筆，才不會忘。
  • 住宿記得填「地址（當地語言）」，給司機看超好用。
  • 確認信（Email PDF、截圖）統一丟進 03-travel-kit/trips/這趟旅程名/ 資料夾，工具內記「確認編號＋檔名」即可。

🧳 打包（三段式）

  • 待買：出發前要買還沒買的。
  • 打包（分人）：每人一鍵套用模板（來自你的「出國行李清單」，含「帶幾件」數量欄），可自行加項目。
  • 出門前檢查：門窗、冷氣、插頭、垃圾、寵物與牠的食物水——出發當天勾完再出門。出發交通每人可設「高鐵／接送／叫車」，高鐵在出發前 3–5 天會於總覽自動提醒買票。

💴 記帳・分帳・匯率

  • 輸入日幣自動換台幣；刷卡的筆可切「台幣」直接記。
  • 分類用點的（甜點、御朱印…），可在「設定」自訂。
  • 每筆可勾「誰參與」（預設全員均分）、選「誰付的」；結算表自動算出誰給誰多少。
  • 匯率：有網路時按「抓即時匯率」（會存起來離線用）；沒網路用手動匯率。

🎁 伴手禮／📶 網卡／🛂 入境 QR／🆘 緊急卡

  • 伴手禮：記誰託買、預算、實花，一鍵帶入記帳。
  • 網卡：選「誰負責買全部人的」，每張卡記方案與開通方式，設定步驟寫一次大家共用。
  • 入境 QR：Visit Japan Web 等連結集中放，QR 截圖放這趟的附件資料夾。
  • 緊急卡：日本官方電話已預填（查證日期 2026-07-20，出發前請再核對官網）；保險、掛失、每人緊急聯絡人自填，可「列印／存 PDF」隨身帶。

📤 分享給同行的人

統計頁 →「產生唯讀分享頁」→ 得到一個 HTML 檔，傳 LINE 群組或放雲端即可。預設只含行程＋訂票住宿；勾「包含記帳與統計」才會有錢的部分。

本工具採「一人維護、大家看」：資料由一台裝置維護，分享頁給大家看，避免多人同時改造成資料打架。

💾 資料與備份

  • 資料只存在你這台裝置的瀏覽器，不會上傳任何地方（含護照等敏感資訊）。
  • 「設定 → 匯出備份 JSON」：換裝置、給家人裝、每趟旅程結束歸檔，都用它。
  • 建議：旅程結束後把備份 JSON 也丟進 trips/這趟旅程名/，跟確認信放一起。
